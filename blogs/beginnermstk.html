Beginner mistakes

5 important mistakes to avoid when creating a website:

1. Not making it mobile responsive

Many beginners do not make their websites mobile responsive. This affects their traffic. 80%. Traffic comes from mobile. If it does not look good on mobile, traffic decreases.

2. Repeated code

Creating multiple files and writing unnecessary code. Common code that is used in many places should be placed in a separate CSS file and linked to it. Doing this reduces the unnecessary length of the code.

3. Ruining the heading structure (incorrect use of h1, h2, etc.)

Beginners make to bold text or use more than 1 h1 tag in the same file. Doing this makes it difficult for Google to understand the structure later.

4.Do not reduce images size.

Beginners tend to put 2-3 mb images directly on the website. This slows down the website, reduces Google speed and increases the bounce rate. Therefore, when placing images, always keep the size between 100 and 200.

5. SEO Meta tag not added

Beginners create websites but do not add SEO meta tags. That is why it is not SEO friendly. For this, use Title, Description, keywords, open graph, favicon.

Other common mistakes:

Using too many colors.
