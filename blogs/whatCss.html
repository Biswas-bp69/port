About CSS

The full form of CSS is Cascading Style Sheets. It makes the website beautiful and attractive. HTML is the skeleton of a person, while CSS is the muscles, clothes, etc.

How to use css?
There are 3 main ways to use CSS.
1. Inline css
(This uses CSS on a single line.)
2. Internal css
(In this, CSS is used within the head tag in the same file.)
3. External css
(For this, a separate CSS file is created and that file is linked inside the head tag in HTML.)
